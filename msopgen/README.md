## ℹ️ 简介

MindStudio Ops Generator（算子工程生成，msOpGen）是算子工程自动生成工具，支持多种类型工程的快速构建，使用户能够专注于算子的核心逻辑与算法实现，避免在项目搭建、编译配置等重复性任务上耗费大量时间，从而显著提升开发效率。

## ⚙️ 功能介绍

目前已支持的功能如下：

| 功能名称                        | 功能描述                                      |
|-----------------------------|-------------------------------------------|
| **算子工程创建**                  | 引导用户按步骤创建标准化的昇腾AI自定义算子工程。                 |
| **算子实现框架生成** | 自动生成Host侧与Kernel侧的算子开发框架代码，支持按步骤完成算子功能实现。 |
| **算子工程编译与部署**               | 在完成开发后，支持一键编译并生成自定义算子安装包（*.run），可部署到昇腾设备。 |
