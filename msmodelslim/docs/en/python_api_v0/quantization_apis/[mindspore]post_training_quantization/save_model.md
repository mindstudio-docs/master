# save_model

## Function

Post-training quantization API, which inserts operators such as AscendQuant and AscendDequant based on the modified graph structure, and saves the model as a quantized offline model file.

## Prototype

```python
save_model(file_name, quantized_model, *input_data, file_format='AIR')
```

## Parameters

| Parameter | Input/Return | Description | Constraints |
| ------ | ------ | ------ | ------ |
| file_name | Input | The path and file name for saving the model. Required. | Data type: string. |
| quantized_model | Input | The quantized model generated by the `quantize_model` API. Required. | Data type: MindSpore model. |
| input_data | Input | The input data of the user network. Required. | Data type: MindSpore Tensor. Must be consistent with the input of the MindSpore model. |
| file_format | Input | The format of the offline model. Required. | Options: AIR and MINDIR. Default value: AIR. |

## Sample

```python
from msmodelslim.mindspore.quant.ptq_quant.save_model import save_model
file_name = "./save_model"
input_data = ms.Tensor(np.random.uniform(size=[1, 3, 224, 224]), dtype=mstype.float32)
calibrate_model = quantize_model(config_file, model, input_data)
save_model(file_name, calibrate_model, input_data, file_format="MINDIR")
```
