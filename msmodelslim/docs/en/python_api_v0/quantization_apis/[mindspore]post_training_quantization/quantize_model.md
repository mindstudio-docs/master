# quantize_model

## Function

Post-training quantization API, which quantizes the graph structure according to the quantization configuration file set by the user. This function inserts weight quantization layers at the layers specified by config_file to complete weight quantization, inserts data quantization layers, and saves the modified network as a new model file.

## Prototype

```python
quantize_model(config_file, model, *input_data)
```

## Parameters

| Parameter | Input/Return | Description | Constraints |
| ------ | ------ | ------ | ------ |
| config_file | Input | Quantization configuration file generated by the `create_quant_config` API, used to specify the configuration of quantization layers in the model. Required. | Data Type: string. |
| model | Input | MindSpore network model with training parameters loaded. Required. | Data Type: MindSpore model. |
| input_data | Input | Input data for the user network. Required. | Data Type: MindSpore Tensor. The shape must be consistent with the input of the MindSpore model. |

## Sample

```python
from msmodelslim.mindspore.quant.ptq_quant.quantize_model import quantize_model
config_file = "./test_config_file.json"
model = SampleModel()
input_data = ms.Tensor(np.random.uniform(size=[1, 3, 224, 224]), dtype=mstype.float32)
calibrate_model = quantize_model(config_file, model, input_data)
```
