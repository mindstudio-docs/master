# Compressor

## Function

Weight compression parameter configuration class, which encapsulates compression algorithms through the Compressor class.

## Prototype

```python
Compressor(config: CompressConfig, weight_path=None, weight=None, quant_model_description=None)
```

## Parameters

| Parameter | Input/Return | Description | Constraints |
| ------ | ------ | ------ | ------ |
| config | Input | The configured CompressConfig class. | Required.<br>Data type: CompressConfig. |
| weight_path | Input | The path to the model weight file to be compressed. | Required. Mutually Exclusive with (`weight` + `quant_model_description`).<br>Data type: String.<br>Remark: Use this method to pass the weight file when the `save_type` parameter output in `save()` is in numpy format. Use `export()` to export the compressed weight file. |
| weight | Input | The sparse quantized weights generated by the quantization tool. | Required. Must be passed together with `quant_model_description`; Mutually Exclusive with `weight_path`.<br>Data type: dict.<br>Remark: Use this method to pass the weight file when the `save_type` parameter output in `save()` is in safe_tensor format. Use `export_safetensors()` to export the compressed weight file. |
| quant_model_description | Input | The quantization weight description file. | Required. Must be passed together with `weight`; Mutually Exclusive with `weight_path`.<br>Data type: dict.<br>Remark: Use this method to pass the weight file when the `save_type` parameter output in `save()` is in safe_tensor format. Use `export_safetensors()` to export the compressed weight file. |

## Sample

- Create a Compressor object using the `weight_path` parameter.

```python
from modeslim.pytorch.weight_compression import CompressConfig, Compressor
compress_config = CompressConfig(do_pseudo_sparse=False, sparse_ratio=1, is_debug=True, compress_disable_layers=None, record_detail_root="./", multiprocess_num=1)
weight_save_path = './quant_weight.npy'  # Modify the path of the weight file to be compressed according to your actual situation.
compressor = Compressor(config=compress_config, weight_path=weight_save_path)
```

- Create a Compressor object using the `weight` and `quant_model_description` parameters.

```python
from safetensors.torch import load_file
import json
# Import weight compression interface
from msmodelslim.pytorch.weight_compression import CompressConfig, Compressor
# Prepare the weight file to be compressed and related compression config. Modify according to your actual situation.
weight_path = "./quant_model_weight_w8a8s.safetensors"       # Path of the weight file to be compressed
json_path = "./quant_model_description_w8a8s.json"          # Path of the description file of the weight file to be compressed
# Use CompressConfig interface to configure compression parameters and return a config instance
compress_config = CompressConfig(do_pseudo_sparse=False, sparse_ratio=1, is_debug=True, compress_disable_layers=None, record_detail_root="./", multiprocess_num=1)
sparse_weight = load_file(weight_path)
with open(json_path, 'r') as f:
    quant_model_description = json.load(f)
# Use Compressor interface, passing in the loaded compression config and the weight file to be compressed.
compressor = Compressor(config=compress_config, weight=sparse_weight, quant_model_description=quant_model_description)
```
