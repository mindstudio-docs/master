## Feature Changes

### 26.0.0

### 8.3.0

Initial release with the following features:

1. Provides core tracing APIs, supporting code execution point marking and scope measurement.

2. Provides domain management APIs for categorizing marks and scopes, enabling data type isolation in multi-tool scenarios.

3. Provides memory tracing APIs, supporting registration and deregistration of memory regions for memory debugging tools.
