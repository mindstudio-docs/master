# Overview

MindStudio Probe (msProbe) is a full-scenario precision toolchain for Ascend. It is designed for precision debugging during model development and help you significantly improve the efficiency of locating model precision problems.

msProbe provides functions such as precision data collection (dump), precision pre-check, training status monitoring, and precision comparison. These functions are designed for different training or inference scenarios, helping you locate precision problems during model training or inference.
