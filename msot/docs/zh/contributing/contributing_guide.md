# 贡献指南

<br>

## 1. 贡献流程和规范

参与开发前，请熟悉贡献流程及相关规范，详细请详见 [《贡献流程和规范说明》](../common/contribute_workflow.md)。

## 2. 贡献方法

本仓库为主仓，不包含具体工具代码。请根据您准备贡献的工具，前往对应工具仓库的 README 查阅详细的《贡献指南》：

| 工具名称            | 功能简介 |                      源码仓库                      |
|-----------------|------|:----------------------------------------------:|
| **msKPP**       | 性能预测 |    [点击查看](https://gitcode.com/Ascend/mskpp)    |
| **msOpGen**     | 工程生成 |   [点击查看](https://gitcode.com/Ascend/msopgen)   |
| **msSanitizer** | 异常检测 | [点击查看](https://gitcode.com/Ascend/mssanitizer) |
| **msDebug**     | 原生调试 |   [点击查看](https://gitcode.com/Ascend/msdebug)   |
| **msOpProf**    | 性能分析 |  [点击查看](https://gitcode.com/Ascend/msopprof)   |
| **msKL**        | 快捷调用 |    [点击查看](https://gitcode.com/Ascend/mskl)     |
| **msOpTuner**   | 参数寻优 |   [点击查看](https://gitcode.com/Ascend/msoptuner)   |
| **msTX**        | 接口扩展 |   [点击查看](https://gitcode.com/Ascend/mstx)   |
| **msOpCom**     | 基础组件 |   [点击查看](https://gitcode.com/Ascend/msopcom)   |
