## Feature Updates

### 8.3.0

#### 1. New Features

This issue is the first official release. The following features are added:

1. Operator feature modeling: The time consumed by operators can be simulated using the APIs provided by msKPP.
2. Operator data transfer specification analysis: The transfer pipeline statistics file and instruction information statistics file can be generated, and you can view the msKPP modeling result.
3. Peak performance analysis: The file instruction pipeline chart and instruction proportion pie chart can be generated, and you can view the msKPP modeling result.
4. Preliminary design of operator tiling: Several optimal tiling policies can be quickly filtered out.
